# ADCx Copenhagen 2026
This is the reference repository on my talk at ADCx Copenhagen 2026, which took place on April 28th at Alboorg University.
The contents here available are shared under the terms of [Creative Common license](https://github.com/rodolfocangiotti/adcx-copenhagen-26/blob/main/LICENSE).

## Post-talk Thoughts
In addition to the information shared during the talk, this work aims to provide a case study about the endless possibilities provided by Cycling '74 Min DevKit. It demostrates that when the Max ecosystem doesn't provide any solution to a given issue or there are still possible improvements, by relying on the Min DevKit is it possibile to produce new Max objects with a streamlined development process.
